1、此工程是音视频自采集demo，基于agora sdk 2.0.2开发。如果使用2.0.2以上版本，可能会需要修改少量代码，例如变量namespace的变动

2、到Agora官网(https://docs.agora.io/cn/Video/downloads)下载agora sdk, 将对应的sdk拷贝到目录 SDK

3、https://pan.baidu.com/s/1BpfbSnuZxC9R2G3DiNW4Vw 下载 .h文件，拷贝到SDK/include

4、本地pc上需要安装directX ,vs2013 搭配 directx june(https://www.microsoft.com/en-us/download/details.aspx?id=6812)
